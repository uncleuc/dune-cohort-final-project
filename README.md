# PyChat

Welcome to PyChat, a Django + Channels real-time chat application built for direct messaging, group conversations, and account management.

## Live URL
https://pychat-kmn3.onrender.com/

## Author

## Highlights
- Real-time chat with Django Channels
- Admin-ready account and API features
- Profile pictures, conversation filters, and friend management
- Production-ready static asset handling with WhiteNoise
